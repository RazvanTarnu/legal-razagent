# Termeni și Condiții pentru RazAgent Video Publisher

**Ultima actualizare:** 23 Iulie 2026

## 1. Introducere
Aceste document reprezintă Termenii și Condițiile de utilizare pentru aplicația "RazAgent Video Publisher" (denumită în continuare "Aplicația"). Aplicația este dezvoltată și operată pentru uz intern (automatizare video AI) și nu este destinată publicului larg.

## 2. Scopul Aplicației
Aplicația utilizează TikTok API (inclusiv Login Kit și Content Posting API) exclusiv pentru a automatiza publicarea de conținut video generat local pe conturile autorizate ale administratorului.

## 3. Restricții de Utilizare
Deoarece aplicația este un instrument intern privat:
- Accesul este restricționat strict la mediul de dezvoltare și producție local (RazAgent_Enterprise_vNext).
- Nu este permisă utilizarea aplicației pentru a distribui spam, conținut malițios sau care încalcă Regulile Comunității TikTok.

## 4. Răspundere
Dezvoltatorul (Răzvan) își asumă întreaga responsabilitate pentru conținutul publicat prin intermediul acestei aplicații. Aplicația interacționează cu serviciile TikTok pe baza credențialelor OAuth2.

## 5. Modificări
Ne rezervăm dreptul de a modifica acești termeni în orice moment, odată cu actualizarea fluxurilor de automatizare din cadrul RazAgent.

## 6. Contact
Pentru orice întrebări tehnice sau legate de conformitate, vă rugăm să ne contactați prin intermediul repository-ului GitHub asociat proiectului.
